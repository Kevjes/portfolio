# Portfolio Kévin Tene - Liquid Glass Design

## 🚀 Description

Portfolio personnel moderne avec un design **Liquid Glass** innovant, développé avec React, TypeScript et des animations fluides. Ce portfolio présente mes compétences en développement Flutter et Web avec une expérience utilisateur immersive.

## ✨ Fonctionnalités

- **Design Liquid Glass** : Interface moderne avec effets de verre liquide
- **Animations fluides** : Transitions et animations avec Framer Motion
- **Responsive Design** : Optimisé pour tous les appareils
- **TypeScript** : Code typé pour une meilleure maintenabilité
- **Performance optimisée** : Chargement rapide et expérience fluide
- **SEO Ready** : Optimisé pour les moteurs de recherche

## 🛠️ Technologies utilisées

- **Frontend** : React 18, TypeScript
- **Animations** : Framer Motion
- **Styles** : CSS3 avec effets Liquid Glass
- **Icons** : React Icons
- **Build** : Create React App
- **Deployment** : Vercel/Netlify ready

## 📦 Installation

1. **Cloner le repository**
```bash
git clone https://github.com/kevin-tene/portfolio-liquid-glass.git
cd portfolio-liquid-glass
```

2. **Installer les dépendances**
```bash
npm install
```

3. **Démarrer le serveur de développement**
```bash
npm start
```

4. **Ouvrir dans le navigateur**
```
http://localhost:3000
```

## 🏗️ Structure du projet

```
src/
```

## 🎨 Design Liquid Glass

Le design Liquid Glass utilise :
- **Backdrop filters** pour les effets de flou
- **Gradients dynamiques** pour les arrière-plans
- **Animations morphing** pour les formes fluides
- **Transparence** et effets de verre
- **Particules flottantes** pour l'immersion

## 📱 Sections du Portfolio

1. **Hero** - Présentation avec photo et liens sociaux
2. **À propos** - Parcours académique et professionnel
3. **Compétences** - Technologies maîtrisées (Flutter, React, etc.)
4. **Expérience** - Timeline des expériences professionnelles
5. **Projets** - Showcase des projets avec détails techniques
6. **Contact** - Formulaire de contact et informations
